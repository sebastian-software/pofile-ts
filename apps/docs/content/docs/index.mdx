**pofile-ts** is a robust TypeScript library for reading and writing GNU gettext PO files. Zero dependencies, full type safety, runs everywhere.

## Features

- **Parse PO Files** — Read PO files from strings with full escape sequence support
- **Serialize PO Files** — Convert PO data back to properly formatted strings
- **Full PO Support** — Headers, comments, flags, plurals, and message context
- **CLDR 48 Plural Data** — 100% compliant for all major languages, just ~3KB
- **ICU Conversion** — Convert between Gettext and ICU MessageFormat
- **Zero Dependencies** — No Node.js APIs required, works in browsers too
- **TypeScript First** — Full type definitions for excellent IDE support
- **Blazing Fast** — Up to 23× faster than alternatives

## Installation

```bash
npm install pofile-ts
```

## Quick Example

```typescript
import { parsePo, stringifyPo } from "pofile-ts"

// Parse a PO file
const po = parsePo(`
msgid "Hello"
msgstr "Hallo"
`)

console.log(po.items[0].msgid) // "Hello"
console.log(po.items[0].msgstr) // ["Hallo"]

// Serialize back to string
console.log(stringifyPo(po))
```

## Performance
